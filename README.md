# Writing Principles

Always communicate outcomes before services.

Instead of:

> We build websites.

Write:

> We build digital experiences that generate trust, qualified leads, and long-term business growth.

Focus on:

- Business impact
- Customer outcomes
- Strategic thinking
- Clarity
- Confidence

Avoid unnecessary technical jargon.

---

# Design Philosophy

Design should communicate trust before creativity.

Principles:

- Simplicity over decoration
- Whitespace over clutter
- Consistency over complexity
- Motion with purpose
- Accessibility by default

Animations should improve understanding, never distract from it.

Every visual element should support business credibility.

---

# Development Philosophy

Development priorities:

1. Maintainability
2. Scalability
3. Performance
4. Accessibility
5. Reusability
6. SEO
7. Visual Quality

Build modular systems rather than isolated pages.

Every component should be reusable.

# AI Working Principles

Every AI assistant working on Nexora should:

## DO

- Think like a business consultant.
- Solve business problems.
- Prioritize clarity.
- Write concise, high-value content.
- Build reusable systems.
- Maintain consistency.
- Follow existing documentation.

## DON'T

- Create generic agency websites.
- Overuse gradients or effects.
- Add unnecessary animations.
- Copy competitors.
- Focus only on services.
- Break the design system.
- Invent information.
